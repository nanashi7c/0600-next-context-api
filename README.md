# 0600-next-context-api

Next.js の Context API を用いたタスク / プロジェクト管理アプリ

## 主な機能

- プロジェクト / タスクの CRUD
- タスク行のインライン編集
- サイドバーによるプロジェクト切り替え
- Chart.js を用いた進捗の可視化

## 技術スタック

- [Next.js 16](https://nextjs.org) (App Router) / React 19 / TypeScript
- [Tailwind CSS v4](https://tailwindcss.com)
- 状態管理: React Context API
- HTTP: axios / 日付: dayjs / アイコン: react-icons / ID: uuid
- Lint: ESLint + husky + lint-staged

## セットアップ

```bash
npm install
```
